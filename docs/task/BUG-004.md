# BUG-004 非 dev 模式不应返回 tool-use 消息

- **status**: completed
- **priority**: P1
- **owner**: claude
- **createdAt**: 2026-03-08 21:00

## 描述

非 dev 模式下 API 仍返回 tool-use 和 system-message 条目到前端，应只返回 user-message 和 assistant-message。

## 修复

- `visibility.ts` — isVisibleForMode 非 dev 只放行 user-message + assistant-message
- `persistence/queries.ts` — VISIBLE_ENTRIES_CONDITION 与 CONVERSATION_MSG_CONDITION 统一
